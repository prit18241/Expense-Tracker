# 💰 Expense Tracker (Android App)

A native Android application designed to help users take control of their personal finances. This app allows users to seamlessly log daily expenses, track their spending history, and monitor their overall balance in an intuitive, easy-to-use interface.

## 🚀 Features
* **Intuitive Dashboard:** A clean UI displaying the current month, total balance, and a quick-add floating action button.
* **Transaction Logging:** Easily add new expenses with details and amounts.
* **History Tracking:** View a complete, scrollable history of past transactions.
* **Offline Storage:** Utilizes a local database architecture (Room/SQLite) to ensure financial data is securely stored on the device and accessible without an internet connection.

## 🛠️ Technology Stack
* **Platform:** Android SDK
* **Language:** Kotlin / Java
* **Database:** Room Persistence Library / SQLite (`AppDatabase`, `ExpenseDao`)
* **UI Components:** XML Layouts, RecyclerView (for transaction history), Material Design components

## 💻 How to Run Locally
1. Clone this repository to your local machine.
2. Open the project in **Android Studio**.
3. Sync the project with Gradle files.
4. Run the app on a connected physical device or via the Android Emulator.
